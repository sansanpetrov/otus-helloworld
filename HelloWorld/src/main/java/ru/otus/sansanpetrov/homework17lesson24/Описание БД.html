Таблица хранения тестов.


Модель «почти» списана с модуля Zephir управления тестированием в Jira


Таблица testcase.

Хранит данные о тесткейсах, кроме результатов выполнения шагов:


id - идентификатор тесткейса;

testname - имя (текстовое поле);

preconditions - условия выполнения тесткейса или подготовительные шаги (текстовое поле. Одна запись-один шаг);

obsolete - устарел тесткейс или актуален (boolean)


Таблица teststeps

Хранит ответы на шаг functionalstep таблицы testcase


id- идентификатор;

functionalstep - шаг тестирования

expectedresult - ожидаемый результат шага

stepresult boolean - результат

testcase_id serial4
